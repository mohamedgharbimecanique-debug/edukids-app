# Dossier Sons (Audio Files)

Ce dossier contient tous les fichiers audio pour les leçons et quiz EDUKIDS:
- Syllabes: ba, pa, ma, ta, da, sa, etc.
- Voyelles: a, e, i, o, u
- Nombres: 1, 2, 3, ..., 10
- Mots clés: bonjour, merci, s'il vous plaît
- Feedback: bravo, essaie encore, bien joué

Format: MP3 ou WAV
Taux d'échantillonnage: 44.1 kHz ou 48 kHz
