# Dossier Images

Ce dossier contient toutes les images éducatives pour EDUKIDS:

## CP - Cours Préparatoire
- Lettres de l'alphabet avec illustrations
- Chiffres 0-10 avec objets dénombrables
- Mots simples illustrés
- Animaux et leurs noms
- Couleurs et formes

## CE1 - Cours Élémentaire 1
- Images pour les leçons de français
- Illustrations pour les maths
- Images du monde (géographie)
- Photos d'objets du quotidien

## CE2 - Cours Élémentaire 2
- Images plus complexes
- Schémas scientifiques
- Cartes et paysages
- Illustrations pour les sciences

Format recommandé: PNG ou JPEG
Dimensions: 400x400px ou 600x600px
